# Bridgescan
> Bridgescan is an explorer for Vite decentralised bridge ViteBridge !
# Features
* Search an address.
* Show bridging details.
* Listen bridging requests.
* Fast & responsive.
# Setup
* Install and open index.html or upload to any website.
